# Rune Hunter

TTRPG design project. JRPG/OSR hybrid.

## Local Development

Start Jekyll preview:
```bash
bundle exec jekyll serve
```
Then open http://localhost:4000

## Git Workflow

```bash
git add -A
git commit -m "description"
git push
```

Pull before starting if changes were made elsewhere:
```bash
git pull
```
